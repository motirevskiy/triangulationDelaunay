Данный код имеет возможность отображать полученную триангуляцию с помощью VTK
Для запуска программы без визуализации:
Находясь в корневой директории
./Compiled\dtWithoutVisual.exe -o output.txt -i Resource\node.txt
Аналогично для запуска программы с визуализацией:
./Compiled\dtWithVisual.exe -o output.txt -i Resource\node.txt
Полученный файл будет находиться в корневой директории - output.txt
Также в случае запуска программы с визуализацией будет создан файл output.png с изображением полученной триангуляции

Для компиляции программы с использованием визуализации необходимо объявить директиву <VISUAL> в файлах Main.cpp и Visualization.cpp
И в случае, если отображение не требуется - удалить директиву <VISUAL>
